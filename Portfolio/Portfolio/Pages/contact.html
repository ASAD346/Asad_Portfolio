Frequently Asked Questions

What is your typical project timeline?

Project timelines vary depending on scope and complexity. A typical web API project takes 2-4 weeks, while a full-stack application can take 6-12 weeks. I'll provide a detailed timeline during our initial consultation.

Do you work on hourly or fixed-price basis?

I offer both hourly and fixed-price arrangements. For well-defined projects, I prefer fixed-price contracts. For ongoing maintenance or projects with evolving requirements, hourly billing works best.

What technologies do you specialize in?

I specialize in .NET technologies including C#, ASP.NET Core, Entity Framework, and Blazor. I also work with SQL Server, PostgreSQL, Azure cloud services, Docker, and modern frontend frameworks like React and Vue.js.

Do you provide post-launch support?

Yes, I offer various support and maintenance packages after project completion. This includes bug fixes, performance optimization, feature updates, and technical support based on your needs.

Can you help with existing projects?

Absolutely! I can help with code reviews, bug fixes, performance optimization, adding new features, or modernizing legacy .NET applications to newer frameworks.
